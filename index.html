We collect the following types of data:

2.2 Purpose of Data Collection

We collect and use this data to:

  • Enable document uploads (photos, PDFs, XLSX, etc.) for employee records
  • Offer organization-specific events such as holidays, birthdays, and other calendar events
  • Improve app functionality and user experience
  • Ensure app security and prevent fraud
  • Generate anonymous usage statistics

2.3 Data Storage and Transmission
